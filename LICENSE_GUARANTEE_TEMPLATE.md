## Крок 2: Форматування документа

1. **Заголовок:**
   - Розмір шрифту: 16pt
   - Шрифт: Times New Roman або Arial
   - Вирівнювання: по центру
   - Жирний

2. **Основний текст:**
   - Розмір шрифту: 12pt
   - Міжрядковий інтервал: 1.5
   - Вирівнювання: по ширині

3. **Підзаголовки:**
   - Розмір шрифту: 14pt
   - Жирний

4. **Відступи:**
   - Верхній/нижній: 2 см
   - Лівий/правий: 2.5 см

## Крок 4: Збережіть як PDF

### Google Docs:
Файл → Завантажити → PDF документ (.pdf)

### Microsoft Word:
Файл → Зберегти як → Тип файлу: PDF

### LibreOffice:
Файл → Експортувати як PDF

## Крок 5: Назвіть файл правильно
